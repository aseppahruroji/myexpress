# API Jadwal Dokter

API kelola master jadwal dokter menggunakan:

- Node.js + Express
- SQLite
- JWT Authentication

## Instalasi

```bash
git clone https://github.com/aseppahruroji/myexpress.git
cd myexpress
npm install
```

## Running

```bash
# jalankan server
node index.js

# generate token
node generate-token.js

# post/get dokter
http://localhost:3000/dokter


# post/get jadwal
http://localhost:3000/jadwal
```
